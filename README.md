# Ejecutar en desarrollo

1. Clonar repositorio
2. Ejecutar 
```
npm install
```
3. tener instalado el nest CLI
```
npm i -g -@nestjs/cli
```
4. levantar la base de datos
```
docker-compose up -d
```

## Stack usado
* NestJs
* MongoDB
